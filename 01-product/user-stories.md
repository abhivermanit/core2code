# User Stories

## Format

```
As a [persona],
I want to [action],
so that [benefit].
```

## Prioritization

| Priority | Label | Description |
|----------|-------|-------------|
| P0 | Critical | Must have for launch |
| P1 | High | Important, but can ship without |
| P2 | Medium | Nice to have |
| P3 | Low | Future consideration |
